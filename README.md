# Weather Suggester

A simple weather application that shows current weather information for any city. It also provides city suggestions as you type.

## Features

- Search for a city and get current weather data.
- Auto-suggestions for city names while typing.
- Displays temperature, humidity, wind speed, and weather icon.
- Fetches user location automatically if allowed.

## Technologies Used

- HTML, CSS, JavaScript
- OpenWeatherMap API
- Fetch API with async/await
